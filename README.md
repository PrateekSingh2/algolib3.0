## ⚡ About

**AlgoLib-v6** is a production-grade interactive algorithm visualizer and logic execution platform. Built by and for developers, it provides an immersive, 3D holographic interface to explore, understand, and visualize complex data structures and logic circuits in real-time. 

Whether you are mastering core computer science fundamentals or competing in high-stakes programming contests, AlgoLib serves as your centralized workspace to visualize, execute, and collaborate.

---

## ✨ Core Features

* **60FPS Interactive Visualizations:** Deep-dive into Data Structures and Algorithms with buttery-smooth animations for Linked Lists, Trees, Graphs, and Sorting arrays.
* **Competitive Programming Arena:** A fully-featured, real-time contest environment with multi-tier algorithmic challenges and global leaderboards.
* **In-Browser Code Execution Engine:** Compile and run logic instantly with support for C++, Java, Python, and JavaScript.
* **Futuristic UI/UX:** A beautifully crafted, alien-inspired interface featuring glassmorphism, dynamic glowing gradients, and cinematic motion physics.
* **Comprehensive Tech Notes Hub:** Production-grade study notes and multi-language code snippets bridging academic theory and practical software engineering.

---

## 🛠 Tech Stack

Engineered for maximum performance and scalability, leveraging modern web technologies:

* **Framework:** React + TypeScript powered by Vite
* **Styling:** Tailwind CSS & shadcn-ui
* **Animation Engine:** Framer Motion
* **Iconography:** Lucide React
* **Backend/Database:** Supabase & Firebase

## 🚀 Local Development

If you want to work locally using your own IDE, you can clone this repo and run the simulator engine on your machine. The only requirement is having Node.js & pnpm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating).

Follow these steps to initialize the matrix:

```sh
# Step 1: Clone the repository using the project's Git URL.
git clone [https://github.com/PrateekSingh2/AlgoLib-v6.git](https://github.com/PrateekSingh2/AlgoLib-v6.git)

# Step 2: Navigate to the project directory.
cd AlgoLib-v6

# Step 3: Install the necessary dependencies.
pnpm install

# Step 4: Start the development server with auto-reloading and instant preview.
pnpm run dev:all
